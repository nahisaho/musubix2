# プロダクト定義

## プロダクト名
MUSUBIX2

## ビジョン
仕様駆動開発（Specification Driven Development）を AI エージェントと統合し、
要件定義からコード生成・テスト・トレーサビリティまでを一貫して管理するシステム。

## 対象ユーザー
- ソフトウェアエンジニア
- AI コーディングアシスタント（GitHub Copilot, Claude 等）
- プロジェクトマネージャー

## 主要機能
1. EARS 形式要件分析・作成・検証
2. SOLID 準拠設計文書自動生成
3. テンプレートベースコード生成
4. テストファースト強制
5. 100% トレーサビリティ監査
6. 9条憲法ガバナンス
7. MCP サーバー統合
8. Agent Skills フレームワーク

## 差別化要因
- 仕様 → 設計 → 実装 → テストの完全自動追跡
- AI エージェントが SDD ワークフローを直接実行可能
- 不変の憲法による品質保証
